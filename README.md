☯️ 山向奇门深层起局排盘系统 (黑金版)

这是一个基于现代 Web 技术构建的单文件、免安装、开箱即用的奇门遁甲排盘工具。应用采用了极具质感的高端“黑金”视觉风格，专为易学研究者、风水师及传统文化爱好者打造。

✨ 核心特性

🌌 沉浸式黑金 UI：以深邃极夜黑为底色，辅以鎏金渐变与明亮的荧光点缀色彩（如天盘红、星门青翠），在暗色环境下提供极佳的护眼与阅读体验。

📊 专业深层排盘算法：

支持按年份、坐山（24山）、朝向、三元（上/中/下）进行自动或手动推演局数（阴/阳遁 1-9 局）。

内盘九宫：精准排列天盘、地盘（含中五寄二宫特殊处理）、八神、九星、八门、暗干及大小神煞。

外盘环绕：首创 24 山、三合五行纳甲、小游年神煞精细外盘环绕布局，高度还原古籍经典排版排布。

🧭 罗经辅助显示：支持输入具体罗盘度数（0-360°），动态联动显示罗盘指针及落区参考。

📥 一键导出高清 PDF：内置截屏与 PDF 生成引擎，一键即可将排盘结果（适配黑金底色）保存为 A4 尺寸的高清 PDF 格式，方便随时查阅、分享或打印存档。

⚡ 零依赖、单文件运行：无需安装 Node.js、无需配置 Webpack/Vite 等繁琐的本地开发环境，只需一个 HTML 文件，双击即可使用。

🛠️ 技术栈

本项目纯前端实现，所有依赖均通过稳定的 CDN 动态加载：

核心框架: React 18 (Standalone UMD)

样式引擎: Tailwind CSS (通过 CDN 即时编译)

代码转换: Babel Standalone (浏览器内实时解析 JSX)

PDF 引擎: html2canvas + jsPDF

🚀 如何使用 (部署与运行)

由于本项目采用的是“单文件”架构，运行极其简单，无需任何服务器支持：

下载项目中的 index.html 文件（或者你自己命名的 奇门排盘黑金版.html）。

在任意电脑或移动端设备上，双击该文件，使用任意现代浏览器（推荐 Chrome、Edge、Safari）打开即可。

注意：导出 PDF 功能由于安全策略，在极少数完全无网络的本地环境下可能会受限，正常联网状态下体验最佳。

📖 操作指南

设置起局参数：

在顶部控制面板调节 立向年份。

选择 坐山 与 朝向（系统会自动联动对应关系）。

选择 罗经三元（上元/中元/下元）。

选择 排盘局数（可保持默认的“系统推演”，或手动强制选择阴/阳遁局数）。

可选：输入具体的 罗盘度数，查看下方图形罗盘的指向。

生成排盘：

调整完参数后，点击大金色的 [生成排盘] 按钮，下方的高端黑金九宫格数据将实时刷新。

导出 PDF：

确认排盘无误后，点击 [一键导出 PDF] 按钮，系统会经过 1-2 秒的渲染，自动将下方的完整九宫排盘图片保存为 PDF 文件下载到你的设备中。
